# Personal Portfolio API

API responsável pelo envio de mensagens do formulário de contato do meu portfólio.

A aplicação recebe os dados enviados pelo frontend, realiza validações, processa a solicitação e envia o e-mail utilizando a API do Resend.

---

## 🚀 Tecnologias

- Node.js
- TypeScript
- Express
- Zod
- Tsyringe (Dependency Injection)
- Resend API
- CORS
- Helmet
- Clean Architecture
- SOLID Principles
